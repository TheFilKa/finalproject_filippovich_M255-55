# ValutaTrade Hub (Final Project)

ValutaTrade Hub — консольное приложение (CLI), имитирующее работу валютного кошелька.

Возможности приложения:
- регистрация и вход пользователей
- управление портфелем и кошельками
- покупка и продажа фиатных и криптовалют
- получение курсов валют из локального кеша с учётом TTL
- отдельный Parser Service для обновления курсов валют  
  (CoinGecko и ExchangeRate-API)

---

## Установка

Для работы проекта требуется установленный `poetry`.

```bash
make install
```

## Запуск
```bash
make project
```
