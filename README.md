# ValutaTrade Hub (Final Project)

Консольное приложение (CLI), имитирующее валютный кошелёк:
- регистрация / вход
- портфель и кошельки
- покупка / продажа валют
- курсы из локального кеша + TTL
- отдельный Parser Service обновляет курсы (CoinGecko + ExchangeRate-API)

## Установка
```bash
make install

## Запуск
make project
